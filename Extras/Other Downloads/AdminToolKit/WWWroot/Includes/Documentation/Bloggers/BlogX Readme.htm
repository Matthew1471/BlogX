Installation Introduction:
Congratulations on your interest in Matthew1471's Blogging solution.
This file should give you the quickest guide to BlogX in a few quick steps.

Installation Requirements:
Internet Information Services
Microsoft Access 2000 Driver or later
FileSystemObject (*Comes With IIS by default*)

Installation Instructions:
Load up your Blogging page e.g. http://yourdomain.com/BlogX/ and login using the username admin and the password letmein.
