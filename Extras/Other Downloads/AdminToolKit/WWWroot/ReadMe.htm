Matthew1471 BlogX AdministrativeToolkit Readme (V.1.0.3)

Installation Introduction:
Congratulations on your interest in Matthew1471's Administrative Blogging solution.
This file should give you the quickest guide to setting up a BlogX server in a few quick steps.

Installation Requirements:
Internet Information Services
Microsoft Access 2000 Driver or later
FileSystemObject (*Comes With IIS by default*)

Installation Instructions:
Step 1. - Extract All The Files In The Zip's "Blogs" To Your Web Server's Directory. e.g. C:\Inetpub\wwwroot\Blogs\
               Make Sure the IUSER account can FULLY WRITE to ALL of the folders and subfolders, the full shabang

Step 2. - Create a Blogs "Database" Folder To A Database Directory. e.g. C:\Inetpub\Database\Blogs\
(The "IUSER_ServerName" must have Write privileges to this file)

Step 3. - Open Up "Includes\Config.asp" e.g. "C:\Inetpub\wwwroot\Blogs\Includes\Config.asp" Edit the paths to your "AppPath", "DatabasePath" (If different from the examples)

That's it... Now load up your Blogging page e.g. http://yourdomain.com/Blogs/ and login using the username "admin" and the password "letmein".

Additional Information:
You can change the configuration at any time by editing the "Includes/Config.asp"
