Matthew1471 WinBlogX Readme (V.1.04.13)

Installation Introduction:
Congratulations on your interest in Matthew1471's Blogging solution.
This file should give you the quickest guide to WinBlogX in a few quick steps.

Installation Requirements:
An Internet Connection
Details For A Matthew1471 Blog Server
Microsoft Internet Explorer 5.5 Or Later
Windows XP (Should Work, But Not Tested On, Other Windows Platforms)

Installation/Usage Instructions:
Step 1. - Double Click "WinBlogX.exe".

Step 2. - Type In The Folder Name Of Your Blog (e.g. for "http://matthew1471.co.uk/Blog/" I type in "Blog").
Step 3. - Type In The Existing UserName & Password Of Your Blog's Admin Panel.
Step 4. - Type In The Servername/Domain Name Of Your Blog (e.g. "matthew1471.co.uk").

That's it... Now click "Ok" (depending on the checkbox, the above information may/may not be stored on your computer)
(*clicking the checkbox ALSO saves the password, so please be carefull where you use this & your choice of password for the BlogX admin panel*).

Additional Information/Points :

  1. If you need to move this to another machine, without installing, you can just copy the core "WinBlogX.exe" file.
  2. It is highly reccomended to press "File->Check Password" to check the details and test the connection to the server for the first time.
  3. WinBlogX uses "Microsoft.XMLHTTP" to connect to "Application.asp" on the server and submit entry information.
  4. WinBlogX shows up in the "Referrers" panel as "WinBlogX".
  5. WinBlogX stores password information in PLAIN TEXT! (Only use a trustworthy PC / Use a unusual unique password just for BlogX).
  6. WinBlogX checks update information against Matthew1471.co.uk's "Refer.asp".

Known Issues:
If you recieve an error running this application, run the Microsoft Visual Basic run-time files package from Microsoft http://support.microsoft.com/default.aspx?scid=kb;en-us;290887
