Matthew1471 Email2BlogX Readme (V.1.00.00 BETA)

Installation Introduction:
Congratulations on your interest in Matthew1471's e-mail Blogging solution.
This file should give you the quickest guide to Email2BlogX in a few quick steps.

Installation Requirements:
An Internet Connection
Details For A Matthew1471 Blog Server
Microsoft Internet Explorer 5.5 Or Later
Windows XP (Should Work, But Not Tested On, Other Windows Platforms)
MSCOMCTL.ocx, MSINET.ocx, msstdfmt.dll
Visual Basic 6 Runtime system file.

Installation/Usage Instructions:
Step 1. - Extract all EXE's and Double Click "Config.exe".
Step 2. - Type In The Folder Name Of Your Blog (e.g. for "http://matthew1471.co.uk/Blog/" I type in "Blog") & Existing UserName & Password Of Your Blog's Admin Panel.
Step 3. - Type In The Servername/Domain Name Of Your E-mail Server (e.g. "matthew1471.co.uk") & Existing UserName & Password for that e-mail account..

Step 4. - Send a plain text e-mail to that account and Double Click "Email2BlogX.exe"

That's it... Now test test test..
(*Email2BlogX saves your passwords, so please be carefull where you use this & your choice of passwords*).

Additional Information/Points :

  1. It is highly reccomended to use the Windows Task Sheduler to execute "Email2BlogX.exe" every 5 minutes.
  2. It also highly reccomended to create a separate e-mail address for Email2BlogX as it does post & delete all e-mails.
  3. It is not recommended to send HTML e-mails, nor e-mails with attachments.

Known Issues:
If you recieve an error running this application, run the Microsoft Visual Basic run-time files package from Microsoft http://support.microsoft.com/default.aspx?scid=kb;en-us;290887
