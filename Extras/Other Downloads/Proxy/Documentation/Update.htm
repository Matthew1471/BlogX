Update From Older Version (Updated):
You can update your Blog to the newest version without loosing any of your data.

1. Replace all your old ASP files with the new ones.
2. Open up the Old Database file, select the records in the "Data" table, copy it to the New database file's table.
3. Open up the New Database file, select each table and manually update the fields in the new database to the old values by refering to the old database.
4. Copy across the comments table, banned entries table..etc if applicable.
