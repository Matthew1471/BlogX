Other Links:
You can change the links in the "Other Links" section by editing "OtherLinks.txt".

The site provides a place to store other links to return under the "Other Links" section,
Simply open up C:\Inetpub\Database\OtherLinks.txt and start editing in this format:

*
TEXT NAME
URL

Then repeat... If the format is incorrect the site will stop processing the "OtherLinks.txt" file and return a simple error.
"OtherLinks.txt" can be useful for system administrators when hosting multiple Blogs.
