Matthew1471 BlogX Proxy Readme (V1.0.0.00)

Installation Introduction:
Congratulations on your interest in Matthew1471's Group Blogging solution.
This file should give you the quickest guide to BlogX in a few quick steps.

Installation Requirements:
Internet Information Services
Microsoft Access 2000 Driver or later
FileSystemObject (*Comes With IIS by default*)
XMLHTTP (Should be installed if IE is installed on your server)

Installation Instructions:
Step 1. - Extract All The Files In The Zip's "WWWroot" To Your Web Server's Directory. e.g. C:\Inetpub\wwwroot\BlogXProxy\

Step 2. - Extract All The Files In The Zip's "Database" Folder To A Database Directory. e.g. C:\Inetpub\Database\

Step 3. - Open Up "Includes\Config.asp" e.g. "C:\Inetpub\wwwroot\BlogX\Includes\Config.asp" Edit the values, like SiteURL and BlogURL (VERY important) and additionally if you need to the "DataFile" & "LinksPaths" (*If Different From Examples*)

Step 4. - Make sure the database can be written to (Ie..The IUSER_ServerName must have Write privallages to this file..) and make sure the Images folder can also be written to (For Pingback, ImageUpload...etc)

Step 5. - Edit your user access list by opening up the database in Microsoft Access and modifying the usernames and passwords in "tblUsers"

That's it... Now load up your BlogProxy page e.g. http://yourdomain.com/BlogXProxy/ and login using the username you specified in the Config (Should be the same as your blogs').

Known Issues:
Q: I get a blank white screen

A: Make sure you are using the latest VBSript runtime environment on your server.

(http://support.microsoft.com/default.aspx?scid=kb;en-us;q290887)
If you are running NAV (Norton Anti Virus) on your server, Turn off "Script blocking".
Norton script blocking is renowned for interfereing with ASP scripts.
