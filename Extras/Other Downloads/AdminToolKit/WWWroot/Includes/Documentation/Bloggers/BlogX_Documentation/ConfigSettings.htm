Matthew1471 BlogX Config Settings

Additional Information:
You can change the configuration at any time by clicking "Config" in the "Admin" panel (Once logged in obviously).

SiteName - What The Site Is Called (Shows Up At The Top)
CommentsURL - If Specified, Will Link "Add comment" To This On Each Item (E-mail address, ForumURL, Chatroom etc)
CookieName - Random Text Value Which Is Stored On Clients Computer & Verified To See If They Have Logged In.
Copyright - Shows Up In Each Pages' Footer.
Description - Shown Just Underneath The Site Name (and as the RSS title).
EntriesPerPage - How Many Entries To Show On One Page.
SiteSubTitle - Shown On The Right
SiteURL - Used To Stop It From Referring Itself, Links To Homepage.
ShowCategories - Will We Let Users Search By Category
SortByDay - Make The Calendar Clickable.
BackgroundColor - The Background Color In The Body Tags.
12HourTimes - AM/PM or 24Hour Clock Times.
Logging - Save Where People Are Coming From In The "Referrers" Table
